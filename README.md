**Personal Portfolio Website**
A modern and responsive personal portfolio website designed to showcase my skills, projects, education, and journey as a Computer Engineering student.

The website is built using HTML, CSS, and JavaScript with a focus on clean UI, responsive design, interactive components, and smooth animations to provide an engaging user experience.

**Features
- Responsive design for desktop, tablet, and mobile devices
- Modern hero section with introduction and call-to-action buttons
- About section highlighting education and background
- Interactive skills section with dynamic effects
- Project showcase with detailed project cards
- Contact form for communication
- Smooth scrolling and section animations
- Responsive navigation bar
- Clean and user-friendly interface

**Technologies Used
- HTML5
- CSS3
- JavaScript
- Git & GitHub

**Purpose
This portfolio website represents my frontend development skills and serves as a platform to showcase my projects, technical abilities, and continuous learning journey.
